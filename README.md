# 拱拱3.0

本项目是拱拱App的前端部分。

3.0迭代选择的技术路线是react-native，没有采用之前的uniapp。

## 使用

### 环境配置

- **react-native 0.75.4**
- **jdk 21**
- **gradle 8.7**
- **android API 34**
- **ndk 26.1.10909125**
- **nodejs 20.17.0**

> [!Note]
>
> 国内下载gradle或插件非常慢，建议设置代理

### 启动

#### 下载依赖

```shell
npm install
```

或者

```shel
yarn install
```

#### 启动项目

```shell
npm run android
```

或者

```shell
yarn run android
```
